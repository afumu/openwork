# OpenWork

OpenWork 是一个由四个部分组成的智能体平台，用于在浏览器中完成编程与应用发布。

目标很直接：部署完成后，组织只需要创建账号并配置模型。用户随后即可登录 Web 应用直接开始工作，无需再额外搭建本地开发环境。

## 目录结构

- `chat/`：面向用户的工作区与聊天界面
- `admin/`：平台管理控制台
- `service/`：API、鉴权、运行时编排与内部桥接
- `pi/`：智能体运行时、工具、工作区、搜索与制品能力

## 本仓库包含

- 按用户隔离的运行时容器
- 运行时打包产物的构建与发布流程
- 基于浏览器的编码工作区
- 工具执行与制品处理能力
- 通用 Web 搜索集成
- 用于集中管理模型配置的内部模型代理

## 本仓库不包含

- 面向单一产品的专用编辑工作流
- 固定的多阶段内容流水线
- 绑定单一使用场景的内置技能包

## 常用命令

```bash
./build.sh
cd admin && pnpm dev
cd chat && pnpm dev
cd service && pnpm dev
cd pi && npm run check
```
