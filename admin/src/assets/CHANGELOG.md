# 项目说明

OpenWork 是一个面向团队与企业私有部署的 Agent 平台模板。

它保留了网页工作台、管理后台、服务层和 PI runtime 四段式骨架，默认支持：

- 多用户账号体系与后台配置
- 浏览器内的对话式编程与工作区操作
- 基于容器的运行时隔离
- 通用工具调用与联网搜索
- 运行时 bundle 发布与切换

## 默认定位

OpenWork 的目标是让部署方在完成模型、搜索和账号配置后，直接把系统交给团队成员使用。用户不需要本地安装开发环境，就可以在网页里完成编码、运行、整理产物和发布程序。

## 模板约定

- 仓库默认不内置行业专用工作流
- 如需品牌信息、支持入口或对外链接，请在部署时替换为你的组织信息
- 如需扩展 Agent 能力，优先在 `service/` 与 `pi/` 中追加通用能力，而不是写死产品流程
