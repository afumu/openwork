# 开源说明 / Open Source Note

OpenWork 是我的第一个开源项目。

这个仓库会持续更新。后续我会根据实际使用、部署反馈和新的平台能力，继续整理代码、完善文档、补充运行时能力，并保持仓库结构尽量清晰、可维护。

OpenWork is my first open source project.

This repository will continue to evolve. Future updates may include code cleanup, documentation improvements, runtime enhancements, deployment refinements, and platform-level capabilities based on real-world usage and feedback.

## 后续更新方式

- 常规代码更新会直接提交到仓库。
- 结构性调整会优先更新 `docs/` 下的说明。
- 涉及前端、后端、运行时或容器脚本的改动，会遵循 [开发风格指南](./development-style.md)。
- 如果后续需要我协助更新，我会按当前仓库规范继续维护。

## Update Policy

- Regular code updates will be committed to this repository.
- Structural changes should be reflected in `docs/`.
- Frontend, backend, runtime, and container changes should follow the [Development Style Guide](./development-style.md).
- When asked to update the project later, I will continue from the current repository conventions.
