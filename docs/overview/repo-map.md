# 仓库地图

## 主要模块

### `chat/`

- 基于 Vue 3 的用户工作区
- 负责聊天界面、工具执行展示、文件与制品界面

### `admin/`

- 基于 Vue 3 + Vite 的管理控制台
- 负责模型配置、搜索配置、运行时设置与平台运维操作

### `service/`

- 基于 NestJS 的后端服务
- 负责鉴权、用量、日志、运行时编排、搜索桥接与模型代理

### `pi/`

- 智能体运行时工作区
- 负责会话、工具、工作区、搜索工具与制品 API
